## 1. 标题

标题层级由 # 个数区分，数量可为 1-6，分别对应一道六级标题

```js
# 一级标题
## 二级标题
### 三级标题
##### 四级标题
##### 五级标题
###### 六级标题
```

快捷键：ctrl+数字（1-6）

## 2. 改变文本颜色

语法：

```html
<span style="color:red;">这里面的内容文字会变成红色</span>
```
